# `goutoujunshi` 联动与语气适配设计

## 目标

将 `girlfriend-reply-coach` 收缩为个性化语气适配器。关系判断、沟通策略和原始话术由 `goutoujunshi` 负责；本 skill 只根据用户本人的说话习惯，以及用户面对当前对象时的表达差异，改写已有内容。

成功标准：相同策略经过本 skill 后更像用户本人，但事实、意图、责任、边界、行动和风险等级不发生变化。

## 职责边界

### `goutoujunshi`

- 判断聊天情境与沟通目标；
- 决定安慰、道歉、解释、协商、边界或停止联系等策略；
- 产出包含必要事实、责任、行动与后续安排的原始话术；
- 处理复杂关系判断和高风险情境。

### `girlfriend-reply-coach`

- 从经授权的聊天记录中提取用户通用语气；
- 提取用户面对指定对象时特有的称呼、句长、语气词、标点、表情、亲密程度和玩笑尺度；
- 忠实改写 `goutoujunshi` 或用户提供的草稿；
- 在主 skill 不可用时生成明确标注的低风险临时回复。

本 skill 不复制、改编或内置 `goutoujunshi` 的理论资料和话术库，也不以“参考军师资料”的方式自行重建关系决策能力。

## 三种运行模式

### 1. 联动模式（默认）

数据流：

`当前聊天与背景 → goutoujunshi 原始回复 → 语义锁定 → 本人语气档案 + 对象差异档案 → 个性化回复`

改写前先锁定不可改变项：事实、沟通目标、责任归属、明确边界、承诺、时间、行动步骤和安全提示。之后只调整表达形式。

### 2. 纯改写模式

用户提供现成草稿并只要求“像我一点”时，不重新分析关系，也不调用军师改变策略。若草稿包含明显的欺骗、威胁、操控、骚扰或虚假承诺，只指出问题并提供不改变合法核心意图的安全表达。

### 3. 降级模式

无法使用 `goutoujunshi` 且用户没有草稿时，可以生成普通临时回复，但必须：

- 明确说明未经过军师情境分析；
- 保持简短、诚实、低风险；
- 不诊断人格或猜测动机；
- 不虚构承诺、经历、礼物、时间或行动；
- 遇到复杂冲突、安全风险、关系重大决定时停止生成策略性回复，并建议补充背景或启用 `goutoujunshi`。

## 个性化数据模型

- `self-voice.json`：用户跨对象稳定的表达习惯。
- `people/<person-id>/profile.json`：当前对象的事实与授权范围。
- `people/<person-id>/voice-delta.json`：用户面对该对象时相对于通用语气的差异。

`voice-delta.json` 只记录表达差异，例如专属称呼、亲密程度、常用梗、可接受的表情和玩笑尺度；不承担对方人格诊断或关系策略判断。多人数据按 `person-id` 隔离，不得混用证据。

## 语义保真规则

改写允许：

- 拆分或合并句子；
- 替换为档案支持的口语、称呼、标点和表情；
- 删除不符合用户习惯的客套和书面表达；
- 在不改变强度的前提下调整自然度。

改写禁止：

- 新增或删除道歉、责任、边界、行动和安全提醒；
- 把确定表达改成含糊表达，或反向改变承诺强度；
- 编造用户不会兑现的内容；
- 为追求“像本人”而复现侮辱、操控、威胁或持续轰炸习惯；
- 把一个对象的专属表达用于另一个对象。

## 输出

默认只给一条可直接发送的个性化回复。必要时补充一条说明：哪些表达来自通用语气、哪些来自对象差异。降级模式必须在回复前标注“临时版本：未经过 `goutoujunshi` 情境分析”。

如果无法在不改变原意的情况下完成改写，应保留原意并指出冲突，不擅自重写策略。

## 错误处理

- 无语气档案：使用自然、简短、无专属称呼的中性口语，并提示可导入聊天记录建立档案。
- 对象未绑定：要求用户指定对象；不得猜测或混用其他人物数据。
- 档案证据冲突：采用高置信度通用习惯，跳过冲突特征。
- 军师输出缺少关键事实：不自行补全，保留占位问题或请求用户提供信息。
- 高风险内容：执行安全边界，不以语气改写绕过限制。

## 验证场景

1. 军师给出完整道歉回复：改写后责任、补救和承诺不变，但语言符合用户档案。
2. 用户提供草稿并要求润色：只改表达，不新增关系判断。
3. 军师不可用：输出带标记的简短临时回复，不作复杂推断。
4. 同时存在多人档案：指定甲时不出现乙的称呼、语气或私有事实。
5. 用户语气档案包含不良习惯：不复现辱骂、威胁、操控或轰炸消息。
6. 输入含现实安全风险：不只生成哄人话术，保留安全提示与求助方向。

## 非目标

- 不训练或微调模型；
- 不建立独立恋爱理论库；
- 不复刻 `goutoujunshi` 的提示词或资料；
- 不保证个性化回复一定获得积极回应；
- 不自动向第三方上传私人聊天记录。
